Co-op prototyping kit

This kit brings together all the Co-op digital's front-end design foundations and components into one place so you can easily test and learn.

This is a kitchen sink of everything. You can copy and paste from the code on this page to create new pages and layouts. This page and its source code are the example.

For full information on how and when to use different foundations and components see the pattern library in the design system.

This is an MVP (minimum viable product) — problems, things not working as expected — email matt.tyas@coopdigital.co.uk

Page layout

The grid is probably the hardest thing to use at first. Below we've added some common layouts you can copy and paste.

Each layout needs to wrapped in a coop-l-row, each column needs a coop-l-column. The columns then span different amounts of the screen at our different breakpoint sizes. To set these add coop-l-[breakpoint-size]-[columnspan].

Breakpoint sizes are:

  • xsmall - 320px
  • small - 480px
  • medium - 750px
  • large - 960px
  • x-large - 1300px

Column spans are numbers 1 to 12 because in total the grid we use consists of 12 equally spaced columns.

That means to have a column width of 7 on a large screen size we would add coop-l-large-7.

The best way for this to make sense is to copy and paste the example grids and mess with the values.

You can also put a grid inside a grid — just paste the code just described inside a grid column.
