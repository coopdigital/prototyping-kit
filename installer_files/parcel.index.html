Co-op prototyping kit

This kit brings together all the Co-op digital's front-end design foundations and components into one place so you can easily test and learn.

This is a kitchen sink of everything. You can copy and paste from the code on this page to create new pages and layouts. This page and its source code are the example.

You can also write you own CSS and HTML to extend this to your needs. Open the main.scss and edit below the comment.

For full information on how and when to use different foundations and components see the pattern library in the design system.

This is an MVP. Problems, things not working as expected — email matt.tyas@coopdigital.co.uk

See the colour palette documentation.

You can use the colour HEX value, the CSS variable or the colour classes directly in your HTML. .coop-u-[colour] will give you a text in your selected colour. .coop-u-[colour]-bg will give you a background colour.

These colours have been contrast tested (and pass at AAA) as background colours with white or grey (#282828) text. Use a colour contrast checker when using any other colour of text.

Page layout

The grid is probably the hardest thing to use at first. Below we've added some common layouts you can copy and paste.

Each layout needs to wrapped in a coop-l-row, each column needs a coop-l-column. The columns then span different amounts of the screen at our different breakpoint sizes. To set these add coop-l-[breakpoint-size]-[columnspan].

Breakpoint sizes are:

  • xsmall - 320px
  • small - 480px
  • medium - 750px
  • large - 960px
  • x-large - 1300px

Column spans are numbers 1 to 12 because in total the grid we use consists of 12 equally spaced columns.

That means to have a column width of 7 on a large screen size we would add coop-l-large-7.

The best way for this to make sense is to copy and paste the example grids and mess with the values.

You can also put a grid inside a grid — just paste the code just described inside a grid column.

Layout helpers

Layout helpers are CSS classes we provide to let you add margin or padding to HTML elements.

The class names are written as coop-u-[padding or margin]-[value].

The values of padding or margin are:

  • x - removes all padding or margin for an element
  • e - eighth (8px)
  • q - quarter (16px)
  • h - half (32px)
  • b - base (64px)

Using the values in this way creates margin or padding on every side of the element. For example coop-u-margin-b will give a 64px margin top, left, bottom and right.

The helper values can also be used directionally by adding:

  • t - top
  • b - bottom

Combining t and h for example coop-u-margin-th will give a top margin of 32px.

Borders

Border classes give a 1px solid border in #C4C4C4 (grey mid) on an element and are written as coop-u-border. Adding t (top), l (left), b (bottom), r (right) will create a border only on that side. For example: coop-u-border-t

Text helpers

Text helpers allow text to be aligned, left right or centred. For example: coop-u-text-centre
